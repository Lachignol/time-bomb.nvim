1. INTRODUCTION					|time-bomb-introduction|

time-bomb.nvim is a timer and Pomodoro manager for Neovim. It displays real-time
floating timers with progress bars and automatic Pomodoro cycles (25/5/25/5/25/15).

Features:
- Custom timers with floating window
- Full Pomodoro cycles with auto-progression  
- 6 progress bar styles
- Pause/resume functionality
- Cycle navigation (next/prev)
- Auto-reposition on resize

3. CONFIGURATION					|time-bomb-configuration|

Available options:

> **enable_default_keymaps** (boolean)
  Enable default keymaps
  Default: `true`

> **keymaps** (table) >
  - `pomodoro_start`: `<leader>tbs` (default) - `:Pomodoro`
  - `timer_custom`: `<leader>tbc` (default)   - `:Timer`
  - `stop_timer`: `<leader>tbe` (default)     - `:StopTimer`
  - `pause_timer`: `<leader>tbp` (default)    - `:PauseTimer`
  - `next_timer`: `<leader>tbn` (default)     - `:NextCycle`
  - `prev_timer`: `<leader>tbb` (default)     - `:PrevCycle`
<

> **pomodoro_cycles** (array of tables) >
  Custom Pomodoro cycles:
  ```lua
  {
    { title = "Work", time = "25", style = "normal" },
    { title = "Short-Break", time = "5", style = "mama-lova" },
    { title = "Work", time = "25", style = "cyberpunk" },
    { title = "Short-Break", time = "5", style = "fire" },
    { title = "Work", time = "25", style = "dots" },
    { title = "Long-Break", time = "15", style = "music" }
  }
  ```
  **Valid styles**: `"normal"`, `"mama-lova"`, `"cyberpunk"`, `"fire"`, `"dots"`, `"music"`
<

> **timer_color** (string)
  Timer color: `"lime"`, `"blue"`, `"black"`, `"gray"`, `"silver"`, `"white"`, `"fuchsia"`
  Default: `"lime"`

> **enable_notification** (boolean)
  Show nvim-notify on timeout
  Default: `false`

> **enable_confirmation** (boolean) 
  Ask confirmation before next cycle
  Default: `false`

Full example: >lua
    require("time-bomb").setup({
      enable_default_keymaps = true,
      enable_notification = true,
      enable_confirmation = true,
      timer_color = "fuchsia",
      keymaps = {
        pomodoro_start = "<leader>p",
        timer_custom = "<leader>t",
        stop_timer = "<leader>x",
        pause_timer = "<leader>z",
        next_timer = "<leader>n",
        prev_timer = "<leader>b",
      },
      pomodoro_cycles = {
        { title = "Work", time = "25", style = "normal" },
        { title = "Short-Break", time = "5", style = "mama-lova" },
        { title = "Work", time = "25", style = "cyberpunk" },
        { title = "Short-Break", time = "5", style = "fire" },
        { title = "Work", time = "25", style = "dots" },
        { title = "Long-Break", time = "20", style = "music" },
      },
    })
<

4. COMMANDS						|time-bomb-commands|

:Timer [N]						*:Timer*
    Start custom N-minute timer.
    Examples: `:Timer 5`, `:Timer 25`, `:Timer` (prompt)

:Pomodoro						*:Pomodoro*
    Start full Pomodoro cycle from current position.

:StopTimer						*:StopTimer*
    Stop current timer immediately.

:PauseTimer						*:PauseTimer*
    Toggle pause/resume current timer.

8. LUA API						|time-bomb-api|

> require("time-bomb").setup(opts)		*time-bomb.setup()*
    Configure plugin.

> require("time-bomb").run_timer(minutes, "TIMER")	*time-bomb.run_timer()*
    Start custom timer.

> require("time-bomb").run_timer("nimp", "POMODORO")	*time-bomb.run_pomodoro()*
    Start next Pomodoro cycle.

> require("time-bomb").stop_timer()		*time-bomb.stop_timer()*
    Stop current timer.

> require("time-bomb").pause_timer()		*time-bomb.pause_timer()*
    Toggle pause/resume.

> require("time-bomb").next_cycle()		*time-bomb.next_cycle()*
> require("time-bomb").prev_cycle()		*time-bomb.prev_cycle()*
    Navigate cycles.
